# [面试题库项目]



## 一、项目概述



[面试题库]旨在[学会next.js+ant design 页面布局，完成登录，菜单授权，题目浏览和管理功能]。







## 二、功能特色



1.功能完整，实现了一个完整的在线登录，权限校验，异常处理，数据处理，题库/题目浏览修改功能。
